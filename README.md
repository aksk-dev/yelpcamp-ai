# Web Development Workspace

This workspace contains practice projects and one main resume-ready project.

## Structure

- `yelpcamp/` - Main Express, EJS, MongoDB, and Mongoose campground review app.
- `learning-projects/` - Smaller Express and Mongoose practice apps.
- `assignments/` - HTML/text assignment files.

For campus placements, polish and deploy `yelpcamp` as the primary project, then mention the learning projects only as supporting practice.

## Recommended GitHub Setup

1. Initialize git from this folder.
2. Commit only source code, docs, `package.json`, and `package-lock.json`.
3. Do not commit `node_modules` or `.env` files.
4. Add screenshots, deployment link, and a clear feature list to `yelpcamp/README.md`.
